Hello World of CI/CD

### To login to VPS
```
ssh -i ~/.ssh/key-2026-03-12-22-03.pem root@133.88.117.56
```
or
```
ssh -i ~/.ssh/key-2026-03-12-22-03.pem jimmy@133.88.117.56
```

### Initial setup in Ubuntu

Update the system
```
sudo apt update
sudo apt upgrade -y
```

Then install Docker
```
sudo apt install docker.io docker-compose -y
```
